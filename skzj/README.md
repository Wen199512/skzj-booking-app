# 首矿之家活动预约系统

一个基于 .NET MAUI 开发的 Android 活动预约应用。

## 应用信息
- 版本: 1.0.11
- 平台: Android 5.0+
- 框架: .NET 10.0 MAUI

## 主要功能
1. 用户登录
2. 活动查询
3. 并行预约
4. 自动更新

## 构建
使用 Visual Studio 2022 打开 skzj.sln 进行构建。
